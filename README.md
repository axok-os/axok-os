# Axok!_OS

Axok!_OS is a GNU/Linux distribution based on **LFS (Linux From Scratch)** focused on learning, simplicity, and understanding how an operating system works from the ground up.

## 📦 About

Axok!_OS is a minimal system built for educational purposes.
The goal is to explore how a Linux system is constructed while keeping everything simple.

## Features

- Built from **LFS (Linux From Scratch)**
  
- Minimal and simple base system
  
- Designeg for learning and experimentation
  
- made by brazilian developers 🇧🇷

## 🎯 Goals

The main goal of Axok!_OS is **Knowledge** - understading how Linux are built and how each component works.

## ⚠️ Disclaimer

Axok!_OS is an experimental project intended mainly for learning and development.
